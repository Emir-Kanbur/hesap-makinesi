# Hesap Makinesi Projesi  
**Bilgisayar Mühendisliğine Giriş** dersi kapsamında ekip olarak geliştirdiğimiz hesap makinesi uygulaması.

## Proje Hakkında
Proje de iki farklı hesap makinesi bulunmaktadır. Temel aritmetik işlemleri gerçekleştirebilen basit bir uygulama ve Temel işlemlerin yerine daha detaylı işlemleri gerçekleştirebilen bir uygulama hesap makinesidir.

Amaç, temel programlama mantığını, fonksiyon kullanımını, bireysel çalışmayı ve  ekip içi iş birliğini geliştirmektir.

##  Temel Hesap Makinesi Özellikleri
- Toplama  
- Çıkarma  
- Çarpma  
- Bölme  
- Hata kontrolü (sıfıra bölme vb.)

## Detaylı Hesap Makinesi Özellikleri
- Üslü Sayı
- Karekök
- Mod Alma
- Faktöriyel
